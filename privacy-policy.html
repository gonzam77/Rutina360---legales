Política de Privacidad

Última actualización: 14 de noviembre de 2025

Esta Política de Privacidad describe cómo se manejan los datos dentro de la aplicación App de Rutinas de Gimnasio (en adelante, "la aplicación").

1. Información que la aplicación NO recopila

La aplicación no recopila, almacena ni transmite ningún dato personal del usuario. Esto incluye, pero no se limita a:

  • Nombre, correo electrónico o identidad
  • Datos biométricos
  • Información financiera
  • Historial de uso vinculado a una identidad personal
  • Ubicación del dispositivo

Todo lo que el usuario configura (rutinas, ejercicios, tiempos, preferencias, etc.) se guarda únicamente en el dispositivo del usuario mediante mecanismos locales como AsyncStorage o bases locales.

2. Datos almacenados localmente

La aplicación puede guardar internamente:

  • Rutinas creadas por el usuario
  • Ejercicios, tiempos de descanso y parámetros de entrenamiento
  • Preferencias de uso

Ninguno de estos datos se envía a servidores externos. El usuario puede borrar la app y estos datos se eliminan automáticamente.

3. Permisos utilizados

3.1 Notificaciones

La aplicación puede solicitar permiso para enviar notificaciones locales, por ejemplo cuando finaliza un temporizador de descanso. Estas notificaciones no involucran datos personales y no requieren conexión a Internet.

3.2 Sonido

La aplicación utiliza sonidos locales para avisos de tiempo. No se accede a micrófono ni se graban audios.

3.3 Funcionamiento en segundo plano

En algunos casos, la aplicación puede ejecutar temporizadores en segundo plano para mantener la funcionalidad del contador de descanso. No se recopila ninguna información durante este proceso.

4. Servicios externos

La aplicación no utiliza servicios de terceros como:

  • Google Analytics
  • Firebase
  • AdMob
  • Servidores propios o ajenos

Si en el futuro se integrara alguno de estos servicios, esta política será actualizada.

5. Seguridad

Dado que la aplicación no recopila datos personales ni utiliza servidores externos, los riesgos relacionados con filtración o acceso no autorizado son extremadamente bajos.

6. Cambios en esta Política de Privacidad

Podemos actualizar esta política ocasionalmente. Recomendamos revisarla periódicamente. La fecha de última actualización aparece al inicio del documento.
